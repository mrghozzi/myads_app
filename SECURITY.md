# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 4.3.x   | :white_check_mark: |
| < 4.3   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within the MYADS app, please report it privately to the author (`mrghozzi`). All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public issues.**
